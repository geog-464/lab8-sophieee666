Path: Python Programmer

Topic: Digitizing Sedimentary Sequences

Project Description: My toolkit will provide a framework for digitizing, plotting and analyzing simple stratigraphic columns.

Project Background: In sedimentary geology, a stratigraphic column is the graphical presentation of a sequence of depositional units. The data necessary to generate these columns is collected in the field and consists of the folowing: unit depth (top/bottom) and unit grain size (minimum/maximum). Traditionally, this data is then represented via simple hand drawings. Compiling these drawings is a painstaking process that would benefit from automation.

Functionality will include:

  • Grain size classification based on the Wentworth scale

  • Depth unit conversion (metric-imperial)

  • Stratigraphic column plotting

  • Graded bedding subsetting
